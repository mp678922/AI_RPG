## 給腳本創作者 ##

這是一個讓Grok運行腳本的範例專案。

以Json形式為基礎而撰寫的RPG腳本。

腳本撰寫者是以VSCode管理多個規則文件來統合整個腳本。

並由自行撰寫的C#程式碼將多份規則文件統一成最終規則書。

### 本專案分類的規則文件區分為： ###

- 以命名 [規則-] 開頭之文件
  可在各種流程重複被使用的文件。
- 以名名 [流程-] 開頭之文件
  為主流程中的其一結構，在主流程文件中有交代AI詳細的運行順序與規則。
- 以名名 [設定-] 開頭之文件
  我將角色設定、地點設定放在這區類。
